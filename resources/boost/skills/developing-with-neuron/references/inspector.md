# Inspector Monitoring

Inspector is the first party monitoring & debugging tool for Neuron AI and Laravel agentic applications.

**Repository:** https://github.com/inspector-apm/inspector-laravel
**MCP Server:** https://docs.inspector.dev/concepts/agentic-integration

Use Inspector when you need to collect metrics and traces about performance and errors of agents, RAG, and Workflow.
Inspector is free for personal use and provides an MCP server to allow coding agent to debug applications with real production data.
